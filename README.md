# graphql-api

This project is an API built with graphql, graphiql, Apollo server, mongoDB, adn reactJS.

## start graphql
nodemon app
The app runs by the port number 4000.

http://localhost:4000/graphql

## Start Client

-npm insall
-npm start

http://localhost:3000/
